# 리스트 노출 조건

## 데이터 필드

| 필드 | 설명 | 필수 여부 |
|------|------|-----------|
| `registered_date` | 등록일 (이슈 등록 시 선택한 날짜) | 필수 |
| `date` | 배포일 | 선택 (null 가능) |
| `status` | 개발 상태 | 필수 |
| `dev_type` | 업무 유형 (퍼블 / 개발 / 퍼블+개발 / 일상) | 선택 |
| `is_event` | 이벤트 여부 | 선택 |

## 개발 상태값

`시작전` / `개발중` / `개발완료`

---

## 캘린더 / 리스트 노출 기준

> 상태(status)와 무관하게 **날짜 기준**으로만 판단

| 조건 | 노출 기준 |
|------|-----------|
| 배포일 있음 | 등록일 ~ 배포일 사이 매일 노출 |
| 배포일이 오늘 이전 | 노출 유지, 타이틀에 취소선 표시 |
| 배포일 없음 | 등록일에만 노출 |

---

## 주간보고 노출 기준

### [금주업무] / [차주업무]

> 동일 기준 적용, 대상 주(week)만 다름

| 조건 | 노출 |
|------|------|
| 배포일 있음 | `등록일 <= 해당 주 마지막날` AND `배포일 >= 해당 주 첫날` (기간이 해당 주에 걸치면 노출) |
| 배포일 없음 | 등록일이 해당 주 안에 있는 것만 노출 |
| `dev_type = 일상` | 제외 |

### [이벤트]

| 조건 | 노출 |
|------|------|
| `is_event = true` | 대상 |
| 배포일(개시일) | 이번 주 시작일 이후 ~ 6주 이내 |
| 이번 주 시작일 이전 배포일 | 제외 |

---

## 회색 처리 (주간보고)

| 조건 | 처리 |
|------|------|
| 배포일이 오늘 이전 | 해당 행 회색 표시 |
| 배포일이 오늘 이후 또는 없음 | 기본 표시 |
